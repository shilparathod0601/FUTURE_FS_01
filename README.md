# Kethavath Shilpa - Portfolio

A modern, professional, and visually stunning portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Hero Section**: Prominent name display with tagline, contact info, and call-to-action buttons
- **About Section**: Professional summary
- **Education Section**: Educational background with animated cards
- **Projects Section**: Project showcase with hover effects and tech stack tags
- **Skills Section**: Animated progress bars for technical skills and soft skills
- **Internship Section**: Internship experience details
- **Certifications Section**: Certification achievements
- **Contact Section**: Contact form and social media links
- **Dark Mode**: Toggle between light and dark themes
- **Responsive Design**: Fully responsive across all devices
- **3D Animations**: Interactive 3D sphere in hero section
- **Particle Effects**: AI-style particle background
- **Smooth Animations**: Framer Motion powered transitions and micro-interactions
- **Sticky Navbar**: Navigation with active section highlighting

## Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Graphics**: React Three Fiber, React Three Drei
- **Particles**: TS Particles
- **Icons**: Emoji and custom styling

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

```bash
npm run build
```

## Deployment

The built files in the `dist` folder can be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.

## Customization

- Update personal information in the respective components
- Modify colors in `tailwind.config.js`
- Adjust animations in component files
- Add more sections or features as needed
